# Azura AI | IT Agency Website

A premium, modern, and responsive IT agency landing page featuring a chatbot widget. Designed for high-end AI, ML, and OCR service providers.

## Features
- **Modern Design**: Dark mode aesthetic with glassmorphism and neon accents.
- **Service Highlights**: Showcases expertise in AI/ML, OCR, LangGraph, RAG, and more.
- **Interactive Chatbot**: Built-in widget for visitor interaction.
- **Responsive**: Fully optimized for mobile and desktop.
- **CI/CD**: Automatic deployment to GitHub Pages via GitHub Actions.

## Technologies
- HTML5 / CSS3 / JavaScript (Vanilla)
- FontAwesome Icons
- Google Fonts (Outfit & Space Grotesk)
- GitHub Actions

## Deployment
This repository is configured for automatic deployment to GitHub Pages. 
1. Push your changes to the `main` branch.
2. Ensure GitHub Pages is set to deploy from "GitHub Actions" in your repository settings.
